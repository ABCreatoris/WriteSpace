Self-hosted fonts (not committed to git — add these files locally under public/fonts/):

  NotoSerifSC-Regular.otf
  NotoSerifSC-Bold.otf
  LXGWWenKaiLite-Regular.ttf
  LXGWWenKaiLite-Medium.ttf

Get Noto Serif SC (OTF):
  https://fonts.google.com/noto/specimen/Noto+Serif+SC → Download → unpack OTF.

Get LXGW WenKai Lite (TTF):
  https://github.com/lxgw/LxgwWenkai-Lite/releases

If a file is missing, the browser falls back to the next family in the font stack.
