Self-hosted fonts (referenced from src/index.css):

  noto-serif-sc-300.woff2
  noto-serif-sc-400.woff2
  noto-serif-sc-500.woff2
  lxgw-wenkai-lite-400.woff2   (and optional 700 if you use bold)

Get Noto Serif SC:
  https://fonts.google.com/noto/specimen/Noto+Serif+SC → Download family → unpack WOFF2.

Get LXGW WenKai Lite:
  https://github.com/lxgw/LxgwWenkai-Lite/releases (or npm lxgw-wenkai-lite-webfont, copy woff2 from node_modules into public/fonts).

If a file is missing, the browser skips that @font-face and uses the Tailwind stack fallbacks.
